This website combines crime data from the LouisvilleKy.Gov open data portal and neighborhood data from Zillow to connect citizens with information about crime in their communities in simple, relevant ways using language and geographies they use everyday.

It also provides a subscription feature, connecting people with what’s going on in their neighborhood on a weekly basis.

Note: This is a January project made by Code for America’s Louisville 2013 fellowship team. Let us know if this is useful, so that we can decide whether to invest more time in it.

Note 2: The site does not report any crime in the Louisville Metro area that falls outside of the neighborhoods presented.
